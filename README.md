# SvelteKit Chat App
A simple chat app made with SvelteKit, Lucia, Socket.io and better-sqlite3

## Features
- Real-time messaging using WebSockets
- Accounts
- Profile pictures
- DMs
- Attachments
- Group chats
- Group chat admins
- Nicknames
